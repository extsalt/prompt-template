# prompt-template
> 
Generate high-yield, single-line summaries and exam insights on [TOPIC NAME] strictly for competitive examinations (GATE / UGC NET / UPSC / State PSC / CUET / University exams).

Each line must:
• Capture one precise concept, rule, limitation, property, or exam trap
• Be factually tight, unambiguous, and technically correct
• Be framed so that it can be directly used as a true/false statement, MCQ option, or assertion-reason pair
• Avoid explanations, examples, or storytelling
• Include common misconceptions and boundary conditions where examiners create traps

Style constraints:
• One insight per line
• Concise, formal, exam-ready language
• No bullet numbering or headings
• No filler words or motivational tone

Coverage constraints:
• Definitions
• Necessary and sufficient conditions
• “If and only if” cases
• Comparisons with closely related concepts
• Edge cases and exceptions

Output only the list of single-line insights.
